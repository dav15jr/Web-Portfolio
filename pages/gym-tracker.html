Workout TrackerWeb App

1st independant project

This is my first independent project built from scratch, from idea to design and implementation. I decided to create a workout tracker as I wanted to build something I am passionate about, fitness. Its purpose is to simplify keeping track of your workouts.

I began by planning what the app should do and how it should work. I then drew what the design and layout of app should be. This was a brief plan and design with a lot of changes and updates implemented.

I started building this project with the basics (Html, Css and JS). Then later started implementing React & TypeScript into the project to improve and enhance its functionality. I moved from local storage to using Firebase for my database and authentication. Helping me improve on my CRUD and RESTful Api building skills.

Along the way I learnt some new libraries like ReCharts, Bootstrap, Sass to improved the app and enhance simplicity while increasing my skillset. The current version works well but has space for additional features in the future. Below you can watch a walkthrough and try the app for yourself.
